# Automated Resume Parser

## 📌 Overview
This project extracts key information from resumes such as name, skills, education, and experience using Natural Language Processing (NLP).

## 🛠 Technologies Used
- Python
- spaCy
- PDFPlumber
- Flask
- PostgreSQL / SQLite

## 🚀 Features
- Extracts structured data from resumes
- Supports PDF and DOC formats
- Stores data in database
- Easy to search candidate details

## 📂 Project Structure
- parser.py → main logic
- app.py → web interface
- database → stored data

## 🎯 Outcome
Built an automated system to process resumes efficiently and reduce manual effort in recruitment.
